# CreativeShulkers

Allows you to open shulkers in your saved hotbars directly from the saved hotbars screen without placing it by right-clicking it. Also supports other containers like barrels or chests.

## Using it on servers

Since the mod is entirely client sided, you can use this on any Minecraft World or Server! The only requirement is that you are in Creative Mode. And don't worry, it is impossible to get banned from the mod since it is indistiguishable from normal saved hotbars.

## Usage

To use it, be in Creative Mode, press E to open the Creative Mode Inventory, go to the Saved Hotbars tab, and right-click a shulker box (or any container) on the screen.
